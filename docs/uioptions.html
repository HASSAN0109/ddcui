Blah blah This option specifies the source of the values shown in the combo box for simple non-continuous (NC) features.
MCCS The values are as specified in the VESA Monitor Control Command Set specification. Note that the valid values can vary by MCCS version.
Capabilities string The values are those listed in the Capabilities string. The interpretation of the values is, if possible, taken from the MCCS spec.
Both The values shown are the union of those in the MCCS spec and those in the Capabilities string. The interpretation of the values is, if possible, taken from the MCCS spec.
