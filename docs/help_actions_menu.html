Actions Menu (2)

The Actions/strong> view has the following choices:
  • Rescan Feature Values - The values shown in the Features can become out of sync with a monitor's actual values. For example, the user can change values using the monitor's On Screen Display, or another program can change the values. This action causes the values to be reread from the monitor.
  • The dialog box controls reporting of various execution statistics, corresponding to those of ddcutil> option --stats.
