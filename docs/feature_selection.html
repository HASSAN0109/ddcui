This dialog selects the features to be reported. The features are specified as a group, and further refined by one or more filters on that group.

The feature groups are:
  • Known - All features defined in the MCCS specification. Note that the defined features vary by MCCS version, as reported by the monitor.
  • Scan - Examine all feature codes.
  • Manufacturer - Examine feature codes in the manufacturer specific range (xe0..xff)
  • Color - Color related feature codes.
  • Profile - Color related feature codes that are saved and restored by loadvcp and savevcp.
The features reported can further be refined by the following filters:
  • Respect Reported Capabilities - Report only those features found in the Capabilities string
  • Include Unsupported Features - Include features that are reported as unspported.
  • Include Table Features - Include features of type Table. Alwasy unset. ddcui does not implement Table type features.
